# Modules

Modules are effectively the 'hacks' that Aoba contains that can be enabled and disabled. 

When enabled, a Module can change or modify additional functionality into the game, such as Flying, Ore XRay, etc...

Modules can be toggled through the ClickGUI, Module Selector HUD, or through chat commands.
